### About project
This project simulates a Kanban Board using Angular CLI and Firebase.

See more about it [here](https://en.wikipedia.org/wiki/Kanban_board).

### Stack used

<p>
    <img width="500" src="./img/angular-stack.png" />
</p>

- Angular CLI
- Firebase
- Angular Material
- Angular SDK
- AngularFire

## How to run this project
```bash
npm install
# and
npm start
```
